数据结构¶

NNS合约的对外统一接口有两个参数，一个是用于接收具体命令的，另一个是接收Object类型的参数列表的，所有合约的接口定义都如下：

public static object Main(string method, object[] args)

因此调用NNS合约的数据结构可描述为：

{
    appCall：合约地址
    method：合约方法
    params：参数列表
}
